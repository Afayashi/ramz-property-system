🗄️ اتصال Supabase
🔑 للحصول على Anon Key:
  1. افتح Supabase → Settings → API
  2. انسخ anon public key (يبدأ بـ eyJ...)
  3. الصقه في الحقل أدناه → اضغط "إعادة الاتصال"
⚠️ المفتاح الحالي sb_publishable_ — إذا لم يعمل استبدله بـ anon public key من Supabase Dashboard

☁️ اتصال Cloudflare Worker
غير متصل
استخدم رابط Worker المنشور مثل https://browser-run.your-subdomain.workers.dev. رابط لوحة التحكم في Cloudflare لا يعمل كـ API مباشر.
🏛️ تكامل منصة إيجار
جاهز للإعداد
اربط بيانات المكتب مع إيجار، وافتح صفحة التسجيل الخارجي عند الحاجة. لا تحفظ كلمات مرور إيجار داخل النظام.
